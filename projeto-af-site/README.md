# O Cookie 🍪

## Tema
Loja fictícia de cookies artesanais. O site apresenta os produtos, lançamentos e edições especiais da confeitaria, além de uma página de contato com formulário e mapa.

## Integrante
- Nelson Capucho Neto - RA: 252079

## Páginas do Projeto
- **Home** (`index.html`) — Carrossel, apresentação, cards de destaque e chamadas para ação
- **Galeria** (`galeria.html`) — Galeria com filtros por categoria (Clássicos, Lançamentos, Edições Especiais)
- **Contato** (`contato.html`) — Formulário completo, informações de contato e mapa incorporado

## Tecnologias Utilizadas
- HTML5 semântico
- CSS3 externo com variáveis e Flexbox
- Bootstrap 5.3.3
- JavaScript (DOM manipulation)
- Google Fonts
- GitHub

## Funcionalidades
- Menu responsivo com Bootstrap Navbar (hambúrguer em mobile)
- Carrossel Bootstrap na Home
- Galeria com filtros por categoria (JavaScript)
- Cards com hover e animações CSS
- Formulário de contato com validação JavaScript
- Mapa incorporado via iframe (Google Maps)
- Botão "Voltar ao topo" com scroll suave
- Animação de entrada com @keyframes
- Efeito gradiente em títulos com `-webkit-background-clip`
- Navbar que reage ao scroll
